## 🎮 在线体验

👉 [点击开始学习](https://v-me-50.github.io/chainsaw-english/)

**多用户支持**：首次使用需设置用户名，每个用户的学习进度独立保存，适合多人共用一台设备。

## 📱 功能特点

- 🔥 电锯人暗黑漫画风格 UI
- 🔊 澳洲口音发音朗读
- 📚 30+ 句地道旅游用语
- 📅 每日打卡系统
- 💾 本地存储学习进度

## 🚀 部署方法

1. Fork 本仓库
2. 进入 Settings > Pages
3. Source 选择 "Deploy from a branch"
4. Branch 选择 "main" / "/ (root)"
5. 保存后等待几分钟，访问 `https://你的用户名.github.io/chainsaw-english/`

## 📝 单词库分类

- 拍照/赞美
- 点餐/购物
- 问路/求助
- 打招呼/寒暄
- 紧急情况

## 🎯 使用场景

准备去澳洲旅游，想学习比教科书更地道的日常表达？每天刷几句，轻松融入当地！
